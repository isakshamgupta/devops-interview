## How to Use

- **Preparing for a specific company?** → Browse `Company-Wise-Questions/` folder
- **Revising a specific topic?** → Browse `Topics-Wise-Questions/` folder
- **Each question is tagged** with `*(CompanyName)*` so you know which company asked it
- **Questions covering multiple topics** are listed in all relevant topic files for completeness
